12.1.1
For all 2010 and subsequent model year vehicles, if a vehicle incorporates a specific engine control strategy to reduce cold start emissions, the OBD II system shall monitor the system to verify the strategy achieves the desired effect (e.g., to achieve accelerated catalyst light-off temperature) and monitor the commanded elements/components for proper function (e.g., injection timing, increased engine idle speed, increased engine load via intake or exhaust throttle activation) while the control strategy is active to ensure proper operation of the control strategy.
