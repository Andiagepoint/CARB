STANDARDIZATION REQUIREMENTS The following standardized functions shall be implemented in accordance with the specifications in SAE J1979 to allow for access to the required information by a scan tool meeting SAE J1978 specifications:
  • Readiness Status
  • Data Stream
  • Freeze Frame
  • Fault Codes
  • Test Results
  • Software Calibration Identification
  • Software Calibration Verification Number
  • Vehicle Identification Number
  • ECU Name
